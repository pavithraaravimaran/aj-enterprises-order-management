# 🛒 AJ Enterprises Order Management System

A Flutter-based Order Management Application developed for **AJ Enterprises** to simplify product management, shop management, and order processing.

The application enables sales representatives to manage products, create customer orders, track delivery status, and generate PDF invoices efficiently using Firebase as the backend.

---

## 📱 Features

### 🏪 Shop Management
- Add new shops
- Store shop name, address, phone number and notes
- Search shops instantly
- View all registered shops

### 📦 Product Management
- Add, edit and delete products
- Upload product images
- Manage MRP, Retail Price and GST
- Search products quickly
- Product details screen
- Categorize products by Brand and Category

### 🏷 Brand & Category Management
- Create brands with images
- Create categories
- Organize products by brand and category
- Search brands and categories

### 🛒 Order Management
- Create customer orders
- Add products with quantity
- Increase/Decrease quantity
- Automatic total calculation
- Order Preview before confirmation
- Edit existing orders
- Cancel orders
- Mark orders as Delivered
- Search orders by shop

### 📄 PDF Invoice
- Export customer orders as PDF
- Professional invoice generation
- Share or print invoices

### 🔥 Firebase Integration
- Cloud Firestore Database
- Firebase Storage for product images
- Real-time updates
